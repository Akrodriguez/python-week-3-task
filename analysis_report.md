# Sales Data Analysis Report

## Project Overview
This project analyzes a simple sales dataset using Python and Pandas to extract meaningful insights such as total revenue and best-selling product.

## Objectives
- Load real-world CSV data
- Clean missing and duplicate data
- Perform basic analysis
- Generate insights for business decisions

## Tools Used
- Python
- Pandas library

## Steps Performed
1. Loaded the dataset using pandas.read_csv()
2. Explored dataset shape, columns, and data types
3. Handled missing values using mean imputation
4. Removed duplicate rows
5. Calculated key metrics:
   - Total Revenue
   - Average Sales
   - Highest Sale
   - Best Selling Product

- Total revenue shows overall business performance
- Cleaning data improves accuracy of results

This project demonstrates how basic data analysis can help understand business performance using simple Python tools.
